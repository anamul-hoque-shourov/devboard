Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element. Example: getElementsByTagName, getElementById, getElementsByClassName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: It includes all the tags, comments of an element, you can show it in the console
innerText: It includes only the texts or values inside an html tag and only shows them in the console.
textContent: It is mostly similar to innerText but whereas innerText ignores the markups, textContent doesn't. But it doesn't render the styles, opposite to innerText.

Question-3: What is event delegation in the DOM?

Event delegation is used to propagate event bubbles in a systemic manner. It is like an regulator that controls the behavior of event bubbling

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event triggered on a child element propagates upward through its parent element in the DOM.
